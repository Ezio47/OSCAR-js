Búsqueda mejorada

La búsqueda mejorada proporciona sugerencias de autocompletar y construcción de consultas a lo largo del proceso.

Una vez que la consulta está completa pulsando ENTER o haciendo clic en el botón Buscar disparará la búsqueda.

Operadores de comparación compatibles

  • >
  • >=
  • <
  • <=
  • <>
  • =
  • LIKE

Operadores lógicos compatibles

  • AND
  • OR

Los valores literales deberán encerrarse entre comillas simples.

dct:abstract = 'unknown'

Se puede utilizar paréntesis para agrupar condiciones.

( dct:abstract = 'unknown AND dc:title='my title') OR dc:title = 'abc'

Caracteres comodín

Los caracteres comodín son compatibles con los valores literales.

  • * (Asterisco) - Se utiliza para sustituir múltiples caracteres.
  • . (Punto) - Se utiliza para reemplazar un solo carácter.

Ejemplos

dct:abstract LIKE 'unknown'
dct:abstract LIKE 'unkn*n'
dct:abstract LIKE 'unkn.wn'
