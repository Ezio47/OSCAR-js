Enhanced Search

Enhanced search provides auto-complete and query building hints along the way.

Supported Comparison Operatators

  • >
  • >=
  • <
  • <=
  • <>
  • =
  • LIKE

Supported Logical Operatators

  • AND
  • OR

Literal values must be wrapped in single quotes. Ex. dct:abstract = 'unknown'

Parenthesis may be used to group conditions. Ex. ( dct:abstract = 'unknown AND dc:title='my title') OR dc:title = 'abc'

Wildcards are also a valid search literal. Ex. dc:title LIKE '*'
